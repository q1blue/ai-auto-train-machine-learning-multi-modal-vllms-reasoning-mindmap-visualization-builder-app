# AI Systems Mindmap

Interactive visualization of AI system architecture with setup instructions and documentation.

## Prerequisites

- Node.js 18 or higher
- Python 3.8 or higher
- Git

## Quick Start

1. Clone the repository:
```bash
git clone <repository-url>
cd ai-systems-mindmap
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Features

- Interactive mindmap visualization
- Comprehensive AI system architecture overview
- Setup instructions and configuration guide
- Documentation section
- Hover descriptions for each component
- Expandable/collapsible nodes

## Project Structure

```
src/
```
